## 开发工具

> 推荐使用HBuilderX进行开发

开发时使用的开发工具为HBuilderX 3.0.4

## 运行方式

1. 打开HBuilderX
2. clone本项目
3. 在HBuilderX工具栏找到运行，运行到真机或模拟器

## 参考文档

1. uni-app官网：https://uniapp.dcloud.io/
2. uView官网：https://uviewui.com/
3. uni-app插件市场：https://ext.dcloud.net.cn/

## 说明
- components目录下的两个文件夹
* public：存放用于多个页面的组件
* private：存放用于单独页面的组件
